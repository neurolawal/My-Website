# Personal Website Starter

This repo contains a static multi-page personal website starter designed around a scientist-first identity rather than a traditional portfolio.

## Files

- `index.html` for the homepage
- `work.html` for projects, builds, experiments, and selected UX work
- `writing.html` for the blog index
- `about.html` for the narrative biography
- `experience.html` for CV / Experience
- `project.html`, `post.html`, and `topic.html` for detail pages
- `site-data.js` for the shared content model
- `app.js` for rendering and filtering
- `styles.css` for layout and visual design

## Edit content

Most content lives in `site-data.js`.

The main models in that file are:

- `siteSettings`
- `projects`
- `posts`
- `topics`
- `experience`

## Preview locally

Open `index.html` in a browser. Since this version does not rely on a build tool, it works as plain static files.
